Literature Survey

Modern office environments require intelligent systems to improve productivity, energy efficiency, employee well-being, and network performance. Existing systems often monitor only one area such as energy, employee workload, or network usage. This project combines multiple intelligent modules into one smart office platform, addressing all these dimensions simultaneously with an integrated, data-driven approach [1][2].

Prior research has shown that isolated solutions — whether energy monitoring, HR tools, or network management — fail to capture the full complexity of a modern smart office [3]. A holistic, multi-modal intelligence layer is required. Our platform integrates three distinct but interconnected intelligent modules to achieve this [4][5].

Research Gap Identified

Most existing office management systems do not combine employee stress detection, task assignment, energy anomaly detection, and bandwidth allocation in one integrated platform. Many systems also lack real-time monitoring, ML-based decision making, and automated alerting.

Unified cross-domain data integration Camera-based real-time stress analysis Automated task adjustment from stress signals ML-driven energy anomaly detection Workload-aware network bandwidth allocation

Research Problem Statement

How can a smart office system use Machine Learning, IoT, and real-time monitoring to improve employee task allocation, reduce energy waste, detect safety risks, and allocate network bandwidth dynamically?

Research Objectives

Main Objective: To design and develop an integrated Smart Office Intelligence Platform that leverages Machine Learning and IoT to optimize task allocation based on stress, monitor energy usage for anomaly detection, and dynamically manage network bandwidth.

Implement camera-based stress detection using CNN/DeepFace Develop stress-aware task assignment logic Create IoT energy monitoring with anomaly detection ML Build workload-aware bandwidth allocation system Integrate all modules into a real-time monitoring dashboard

Methodology

Module 1 — Task Allocation: Admin sends camera request → Employee accepts/rejects → Webcam captures face image → ML predicts stress (Low/Medium/High) → Workload assigned accordingly → Task auto-cycles on completion.

Module 2 — Energy Optimization: ESP32 nodes collect power/voltage/current/frequency + temperature/humidity/gas readings → MQTT/TLS transmission → Feature engineering → ML anomaly detection → Alerts via n8n + Twilio.

Module 3 — Bandwidth Allocation: Endpoint agents collect CPU/RAM/network usage → ML classifies workload tier → Bandwidth allocated → SD-WAN selects optimal path → Live dashboard.

Research Components

Our Research Components

Three intelligent, interconnected modules forming a unified smart office platform.

01

Camera-Based Stress Detection & Task Allocation

Detects employee stress level using webcam-based facial analysis and assigns suitable tasks based on stress level and workload category.

Admin requests employee webcam access
Face image captured and analyzed by ML model
Stress level classified: Low / Medium / High
Task assigned based on predicted stress tier
Next task auto-assigned on completion
02

IoT-Based Office Energy Optimization

Monitors office energy and environmental conditions using ESP32 IoT nodes and detects power/air-quality anomalies using ML models.

ESP32 sensors collect power, temperature, gas data
Secure data transmission via MQTT/TLS
Random Forest detects power anomalies
Isolation Forest detects air-quality risks
Automated alerts via n8n + Twilio
03

AI-Driven Dynamic Bandwidth Allocation

Monitors employee workload and application activity, classifies workload intensity using ML, and dynamically allocates bandwidth.

Agents collect CPU, RAM, network usage
ML classifies workload: Low / Medium / High
Bandwidth regressor predicts optimal Mbps
SD-WAN controller selects best network path
Live dashboard shows tier, bandwidth & WAN path
